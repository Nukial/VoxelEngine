## GDVoxelPlayground

Repository: https://github.com/JorisAR/GDVoxelPlayground  
Author: JorisAR  
License: MIT License  
Copyright: Copyright (c) 2025 JorisAR

GDVoxelPlayground is a Godot-based voxel playground that implements a cellular automata voxel playground rendered using ray marching. Its documented features include a ray marcher optimized using a brick map and cellular automata with multiple materials.

VoxelEngine acknowledges GDVoxelPlayground as an important technical reference and inspiration source for exploring GPU-style voxel playground ideas, including ray-marched voxel rendering, brick-map acceleration, and cellular automata material behavior.

VoxelEngine is a separate Unity/C#/HLSL research prototype with its own project structure, implementation choices, and license terms. It is not affiliated with, endorsed by, or maintained by JorisAR.

If any code, shader logic, assets, or substantial implementation details from GDVoxelPlayground are incorporated into this repository, the MIT license notice below must be preserved for those portions.
